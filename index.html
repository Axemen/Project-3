The Data

scraping and cleaning

To begin this project, we sought out a valid source for whisky reviews, one that not only held a comprehenisve and diverse catalog of whiskys but also supplied information on each in a consistent manner. After searching through a few options, we settled on using Whisky Advocate as the main data source for the following points: distillery, label, year, age, style, category, review points and description.
